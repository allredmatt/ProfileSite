My profile page I wanted to be quick light and just HTML and CSS, all icons to be SVG where appropriate, so that it loads fast.

Strengths of this project:

  • Fast
  • Responsive, developed for both mobile and desktop experience
  • Was quick to develop, so didn't take up too much time

Weaknesses of this project:

  • Design not a crisp as possible, didn't want to over design and spend free time on this
  • Uses external blog, in future will add own blog site to this
  • Needs adding some accessibility work, missing aria attributes, contrast of text isn't clear enough
